一个下载图片的node模块

安装：

npm install getpic

建立images文件夹，和测试文件 test.js：

var getimg=require('getpic');

var url = process.argv[2];

var path = process.argv[3];

path= path || 'images';

getimg(url,path);

运行：

node test.js http://www.codetyphon.com images
